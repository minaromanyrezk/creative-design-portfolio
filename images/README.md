## Purpose
This folder contains design assets and reference images for use with Gemini AI and other visual AI tools. These assets serve as visual references for:

- Design analysis and feedback
- Visual style guides
- Inspiration and mood boards
- Portfolio showcase
- Project previews

## Folder Structure

### `/brand-identity`
Brand system assets including logos, color palettes, typography samples, and visual guidelines.

### `/social-media`
Social media design templates and optimized assets for different platforms (Instagram, Facebook, LinkedIn, etc.).

### `/campaigns`
Marketing campaign visuals, including teaser designs, launch assets, and promotional materials.

### `/ai-generated`
AI-assisted design concepts and variations for analysis and style reference.

### `/product-design`
Product photography, mockups, and packaging design references.

## Usage with Gemini

### Direct URL Reference
You can reference images directly in Gemini using their raw GitHub URLs:
```
https://raw.githubusercontent.com/minaromanyrezk/creative-design-portfolio/main/images/[file-path]
```

### How to Use
1. Upload images directly to Gemini's interface, or
2. Provide the GitHub raw URL for Gemini to analyze
3. Ask Gemini for design analysis, feedback, or creative suggestions

## Notes
- All images are organized by project type for easy reference
- File names follow descriptive naming conventions
- Keep this folder updated with latest design work
- Use high-resolution versions for analysis and feedback
